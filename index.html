Calculator Info

  • For Regular classes: A = 4.0, B = 3.0, C = 2.0, ...
  • For Honors/AP/Dual Enrollment classes: A = 5.0, B = 4.0, C = 3.0, ...
  • A semester of a class grants you 5 credits. Dual Enrollment Partnership classes with El Camino are 10 credits each (the equivalent of a year's credits).
  • Weighted GPA = Σ(Grades * Credits) / Σ(Credits)
  • PE/Sport classes are not considered in your weighted GPA.
